# Instrcciones para programa Metro

## 1° Prompt
Suponiendo que quiero automatizar todo el proceso para que siga el siguiente orden:

1. Pasar información de las lineas de un archivo .csv a Python.

2. Crear nodos de un grafo que represente las estaciones.

3. Crear las uniones y escribir las distancias que hay entre estaciones.

4. Mostrar con una interfaz el grafo generado.

5. Integrar algoritmos de Dijkstra y A* para que el usuario pueda escoger uno de los 2 para realizar el calculo de la distancia entre estaciones.

6. Realizar el algoritmo pedido con las estaciones que se les pida hacer el calculo entre estaciones.
